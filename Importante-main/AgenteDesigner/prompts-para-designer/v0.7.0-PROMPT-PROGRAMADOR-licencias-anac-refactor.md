# v0.7.0 — Refactor: Adaptar Perfil Piloto al nuevo modelo LicenciaANAC

**VERSIÓN:** v0.7.0  
**SLUG:** licencias-anac-refactor  
**DEPENDENCIAS:** v0.6.0, v0.6.1  
**PRIORIDAD:** alta — la vista actual envía datos con el modelo viejo (Licencia SW) al backend que ahora espera LicenciaANAC

---

## Descripción del cambio

El backend separó **Licencia** (software: nombre, numLicencia, version, compra) de **LicenciaANAC** (piloto: fechaVencimientoCma, fechaEmision, caducidad, imagenCma, imagenCertificadoIdoneidad). Además se eliminó el endpoint separado de CMA (`/mi-perfil/cma`); ahora el CMA es **por licencia ANAC** (cada licencia tiene su propia `fechaVencimientoCma` e `imagenCma`).

También se agregó **foto de perfil** (`/mi-perfil/foto-perfil`).

La vista `PerfilPilotoView.vue` y el API client `mi-perfil.js` usan el modelo viejo y hay que adaptarlos.

    method: 'PUT',

async function onSaveLicencia() {

  const body = {
    fechaVencimientoCma: f.fechaVencimientoCma || null,
    fechaEmision: f.fechaEmision || null,

async function onCmaImageSelected(event) {
  const file = event.target.files?.[0]
  if (!file) return
  if (file.size > 10 * 1024 * 1024) { showToast('Máximo 10 MB.'); return }
  licImageModal.value.uploadingCma = true
  try {
    await subirImagenCmaLicencia(licImageModal.value.licencia.id, file)

  finally {
    licImageModal.value.uploadingCma = false
    if (licCmaFileInput.value) licCmaFileInput.value.value = ''
  }

async function onCertImageSelected(event) {
  const file = event.target.files?.[0]
  if (!file) return

  licImageModal.value.uploadingCert = true
  try {

    if (blob) licImageModal.value.certUrl = trackUrl(URL.createObjectURL(blob))

  finally {
    licImageModal.value.uploadingCert = false
    if (licCertFileInput.value) licCertFileInput.value.value = ''
  }
}

## Referencia rápida: modelo viejo vs nuevo

| Concepto | ANTES (viejo) | AHORA (nuevo) |
|---|---|---|
| CMA | Endpoint separado `/mi-perfil/cma` | Campo `fechaVencimientoCma` dentro de LicenciaANAC |
| Imagen CMA | `/mi-perfil/cma/imagen` | `/mi-perfil/licencias/{id}/imagen-cma` |
| Licencia (perfil piloto) | Campos SW: nombre, numLicencia, version | Campos ANAC: fechaVencimientoCma, fechaEmision, caducidad |
| Imagen licencia | `/mi-perfil/licencias/{id}/imagen` (una) | Dos: `.../imagen-cma` + `.../imagen-certificado-idoneidad` |
| Foto de perfil | No existía | `PUT/GET /mi-perfil/foto-perfil` (nuevo) |
